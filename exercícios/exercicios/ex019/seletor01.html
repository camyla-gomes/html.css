Criando Sites com HTML e CSS

Apredendo HTML

HTML básico

HTML (Linguagem de Marcação de Hipertexto) é o código que você usa para estruturar uma página web e seu conteúdo. Por exemplo, o conteúdo pode ser estruturado em parágrafos, em uma lista com marcadores ou usando imagens e tabelas. Como o título sugere, este artigo fornecerá uma compreensão básica do HTML e suas funções.

HTML intermedíario

HTML Intermediário (Hypertext Markup Language) é projetado para uma linguagem com recursos mais avançados e é direcionado para aqueles que já têm alguma experiência na criação de páginas usando HTML. Tem como objetivo melhorar a capacidade dos membros de criar sites com mais recursos e opções. Também oferece várias práticas recomendadas, como HTML com integração CSS (Cascading Style Sheet), links, formatação de teto avançada, parágrafos e quebras de linha, cabeçalhos, listas, etc. Mostra os erros e precauções mais comuns ao construir um site

HTML avançado

HTML Avançado (Hypertext Markup Language) é projetado para uma linguagem com recursos mais avançados e é direcionado para aqueles que já têm alguma experiência na criação de páginas usando HTML. Tem como objetivo melhorar a capacidade dos membros de criar sites com mais recursos e opções.

Apredendo CSS

CSS básico

CSS básico apresenta tudo que você precisa para começar. Responderemos às perguntas do tipo: Como mudo meu texto para preto ou vermelho? Como fazer o meu conteúdo em que posso fazer na tela? Como decoro minha página com imagens e cores de fundo?

CSS intermedíario

O CSS é uma linguagem de estilização, ou seja, ela é utilizada para definir como os documentos escritos na linguagem de marcação (HTML ou XML) devem ser apresentados em termos de formatação e layout.

CSS avançado

CSS Avançado – Posicionamento você irá aprender a partir dos fundamentos como posicionar os elementos em uma página. O foco é passar de maneira exaustiva os fundamentos das propriedades e como elas agem no layout do site.
